## How to clear demo checklist content?


Tap Settings button in top left corner to open settings panel, then tap Purge Database button.
Confirm by tap Delete button to all checklist content on confirmation dialog. Warning! This action is not reversible.

## How to clear/set NoteBar note?

<Divider/>

Long press on NoteBar, to open note editor. Now you can clear the existing note by tapping Clear button.
Or you can set a new note by typing a new note text. Tap Save button to save the note.
